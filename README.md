# Axe Playwright Accessibility CLI

A CLI tool for automated accessibility testing using Playwright and Axe.

## Features
- Multi-page scanning
- JSON + HTML reports
- Dashboard summary
- CI/CD ready

## Usage
axe-audit scan

## Tech Stack
- Playwright
- Axe-core
- Node.js

Step 1: Install Dependencies
npm init -y
npm install playwright axe-core fs-extra chalk commander
